# Ariadna — Project Context

Ariadna is a browser-first prototype of a 3D / hierarchical thought-mapping interface.

## Goal of the MVP
Build a minimal 3D board where a user can:
- see nodes in 3D space
- click a node
- open a simple text card
- move through the scene
- later connect nodes manually

## Product meaning
This is not a game.
This is not a flashy 3D demo.
This is a thinking interface.

The interface should feel:
- spatial
- clear
- calm
- modular
- readable

# Product Reference
Ariadna should move toward a product feeling closer to Think Machine (thinkmachine.com) in spirit, but not as a clone.

Reference qualities:
- calm and clean visual language
- graph-first thinking
- text and structure should feel connected
- 3D space should support understanding, not just visual spectacle
- AI is a later layer built on top of the graph, not a replacement for the graph

## Current priority
Prototype first.
Readable architecture first.
One layer at a time.
