# Добавление навигации камеры (Orbit Controls)

Мы добавим базовое управление камерой в сцену. Это позволит вам вращать (левая кнопка мыши), панорамировать (правая кнопка) и масштабировать (колесико) 3D-сцену, чтобы можно было осматривать тестовую сферу со всех сторон. 

## Цель (Goal)
Добавить стабильную 3D-навигацию к существующему холсту, не нарушая имеющуюся архитектуру и не добавляя сложной кастомной логики обработки мыши.

## Критерии приемки (Acceptance Criteria)
- В браузере можно взаимодействовать со сценой с помощью мыши.
- Знакомое поведение `OrbitControls` из 3D-редакторов.
- Исходная вращающаяся сфера, освещение и размеры окна остаются неизменными.

## Затронутые файлы/модули
- `package.json` — установка зависимости `@react-three/drei`.
- `src/Scene.tsx` — внедрение компонента `<OrbitControls />` внутрь `<Canvas>`.

## Одобренные зависимости
- **`@react-three/drei`** — официальный, легковесный пакет-помощник для `@react-three/fiber`, избавляет от императивного грязного кода кастомных Orbit-хуков.

## Что НЕ включено
- Выделение узлов (`raycaster` для кликов).
- Перетаскивание объектов.
