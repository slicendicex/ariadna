# Статические 3D рёбра (Static Edges)

Мы добавляем визуальные линии между узлами, чтобы сцена выглядела как минимальный граф.

## Критерии приемки (Acceptance Criteria)
- Создан новый компонент `src/scene/Edge3D.tsx`.
- `Edge3D` принимает `from` и `to` как `[number, number, number]` и рисует линию.
- `Scene.tsx` содержит `MOCK_EDGE_DATA` на уровне модуля (массив `{ id, fromId, toId }`).
- Если edge ссылается на несуществующий узел — он пропускается (safe skip).
- Рёбра рендерятся через `.map()` в `<Canvas>`, параллельно узлам.
- Цвет линий — спокойный серый. Никаких новых зависимостей.

## Затронутые файлы
- `src/scene/Edge3D.tsx` `[NEW]` — компонент линии между двумя точками.
- `src/Scene.tsx` `[MODIFIED]` — добавлен `MOCK_EDGE_DATA` и рендер `<Edge3D>`.

## Что НЕ включено
- Интерактивность рёбер (клик, hover).
- Создание рёбер через UI.
- Новые зависимости.
