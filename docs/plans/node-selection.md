# Выделение узла по клику (Click Selection)

Мы добавляем возможность выделения (selection) конкретной сферы-узла с помощью левой кнопки мыши. Для этого мы задействуем встроенную систему событий `@react-three/fiber` (Raycasting).

## Цель (Goal)
Внедрить локальное состояние выбора в компонент сцены и передавать узлам флаг их выделенности. Визуально выделенный узел будет отличаться от остальных.

## Критерии приемки (Acceptance Criteria)
- В `src/Scene.tsx` добавлен стейт `selectedNodeId` для хранения идентификатора выделенного узла.
- Интерфейс `Node3DProps` расширен параметрами `isSelected: boolean` и `onClick: () => void`.  
- При клике на `<Node3D>` вызывается `onClick` событие, которое через `e.stopPropagation()` перехватывает клик и меняет `selectedNodeId` в `Scene.tsx`.
- При клике "в пустоту" (по холсту) `selectedNodeId` обнуляется через `onPointerMissed={() => setSelectedNodeId(null)}` в `<Canvas>`.
- Если узел выделен (`isSelected === true`), он визуально масштабируется (`scale={[1.2, 1.2, 1.2]}`).

## Затронутые файлы/модули
- `src/Scene.tsx` `[MODIFIED]` — хранит стейт выделения `selectedNodeId` и передает новые пропсы. Отвечает за обнуление выделения при клике в пустоту.
- `src/scene/Node3D.tsx` `[MODIFIED]` — принимает `isSelected` и `onClick`, реагирует визуально через увеличение масштаба.

## Что НЕ включено
- Глобальный стейт-менеджер (Zustand по-прежнему не добавляется). 
- Стейт выделения остается локальным в `Scene.tsx`.
- 2D-карточка с информацией по-прежнему отсутствует.
- Эффекты обводки (outline) или свечения.
