# Создание минимальной 3D-сцены Ariadna

Мы добавим базовое 3D-окружение в текущее React-приложение. Это первый шаг к пространственному интерфейсу проекта.

## Цель (Goal)
Отобразить пустую 3D-сцену, на которой находится хотя бы один тестовый трёхмерный объект, чтобы убедиться, что рендеринг и камера работают корректно.

## Критерии приемки (Acceptance Criteria)
- В браузере рендерится 3D-холст (canvas) на весь экран.
- Камера корректно отцентрована.
- По центру сцены отрисовывается простой геометрический примитив (например, куб), чтобы сцену было видно.
- Добавлено минимальное освещение (ambient/directional light).
- Приложение запускается без ошибок.

## Затронутые файлы/модули
- `package.json` — установка новых библиотек.
- `src/App.tsx` — интеграция холста.
- `src/Scene.tsx` `[НОВЫЙ]` — компонент или модуль, отвечающий за логику 3D-окружения.
- `src/index.css` — стили для растягивания canvas на весь экран аппаратно.

## Одобренные Зависимости
1. `three` — базовый 3D-движок.
2. `@react-three/fiber` — мост между React и Three.js. Избавляет от написания ручного цикла рендеринга и обработки resize.
3. `@types/three` — типы для TS.

## Что НЕ будет включено (утвержденные ограничения)
- Навигация камерой (orbit controls добавятся следующим шагом).
- Карточки узлов, текст в 3D.
- Перетаскивание объектов, создание связей.
- Любой дополнительный UI.
